TO FORK A REPOSITORY: 1.Navigate to the repository which you want to fork. 2.Click on the FORK option on the top right corner. 3.Now the forked repo appears with your username . 4.Click on Code and copy the URL. 5.Open gitbash in terminal and configure your git profile details to which you want to fork. 6.Change the directory to where you want to store the forked repo in local. 7.Initialize git. 8.Type git clone and paste the copied URL and press enter. 9.This creates a local copy of forked repository.
